# .github

This repository holds organization-wide defaults for all Cerify.ai repositories.

For more about Cerify.ai, visit our [profile](https://github.com/Cerify-Systems) or [cerify.ai](https://cerify.ai).

## What's in here

- **CONTRIBUTING.md** — our contribution workflow (forking, branching, pull requests)
- **CODE_OF_CONDUCT.md** — expected standards of behavior
- **SECURITY.md** — how to report a security issue
- **SUPPORT.md** — where to go for help
- **PULL_REQUEST_TEMPLATE.md** — default checklist applied to new pull requests

These files automatically apply as defaults to any repository in this organization that doesn't have its own copy of the same file.

## Note

This repository is public so these defaults can be applied organization-wide. It does not contain, link to, or reference any private repositories, source code, or internal systems.
